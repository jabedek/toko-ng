Firefox

To be able to use Web Speech API in Firefox, go to about:config and switch these flags to true:
  • * media.webspeech.recognition.enable
  • * media.webspeech.recognition.force_enable
  • * media.webspeech.synth.enabled
Are you aware that you have to enable the access yourself?
